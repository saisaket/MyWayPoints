Welcome to the mywaypoints web application file
The application is used to retrieve weather along the route you want to travel 

GETTING STARTED:
These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. 
See deployment for notes on how to deploy the project on a live system.

REQUIREMENTS:
install eclipse photon with jre version - jre1.8.0_181
install apache tomcat server v9.0
install the tomcat server onto the eclipse ide

INSTALLING:
import the project on to the eclipse server

Run the project on the eclipse IDE by simply right clicking on the project and choose option run as on Run on Server.
